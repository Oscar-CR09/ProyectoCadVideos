Bienvenidos

Oscar Cervantes, Desarrollador Full Stack

Desarrolador Web

Desarrollador Full Stack, con experiencia en el uso de Java (8, 11, 17) y Python complementando con aws por parte de la nube. Experto en la gestión de entornos virtuales y paquetes, así como en el desarrollo y prueba de código. Altamente capacitado en el uso de Git y GitHub para el control de versiones y la colaboración en proyectos. Experiencia en la creación y gestión de imágenes y contenedores de Docker. Motivado para resolver problemas complejos y crear soluciones eficientes. Apasionado por la tecnología y el desarrollo de aplicaciones web escalables. Capaz de aprender rápidamente y adaptarse a nuevas tecnologías.

En este blog comparto mis proyectos de desarrollo web, desde aplicaciones simples hasta soluciones complejas. Aquí encontrarás ejemplos de código, tutoriales y recursos para aprender sobre desarrollo fullstack. Mi objetivo es inspirar a otros desarrolladores y compartir conocimientos sobre las últimas tecnologías en el mundo del desarrollo web.
